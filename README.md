## How to use this app

```bash
$ npm install
```

Development on an emulator:

```bash
$ ionic cordova platform add ios
$ ionic cordova run ios
```

Development on browser:

```bash
$ ionic cordova platform add ios
$ ionic serve
```

Then, to build it:

```bash
$ ionic cordova build ios
```

Substitute ios for android if not on a Mac.
